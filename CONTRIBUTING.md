# 貢献について

プロジェクトへの貢献を歓迎します！Pull Requestはお気軽に送ってください。

## 始め方

1. リポジトリをクローンします
2. Rustのツールチェーン（cargo）をインストールします
3. 提出する前にテストを実行してください

```bash
cargo test
```

## フォーマット

コードが正しくフォーマットされているか確認してください：

```bash
cargo fmt -- --check
```

## コミットメッセージ

可能であれば、Conventional Commitsに従ってください（例：`feat: 検索機能の追加`, `fix: テーブル表示の修正`）。
