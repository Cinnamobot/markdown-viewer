# Markdown Viewer Workbench

このファイルはmarkdown-viewerの機能を包括的にテストするためのワークベンチです。
さまざまなMarkdown要素を試し、レンダリングや機能の課題を発見します。

## コードブロックテスト

### 言語指定あり

```rust
fn main() {
    println!("Hello, world!");
    let x = 42;
    if x > 40 {
        println!("x is greater than 40");
    }
}
```

```python
def hello_world():
    print("Hello, world!")
    x = 42
    if x > 40:
        print("x is greater than 40")

hello_world()
```

```javascript
function helloWorld() {
    console.log("Hello, world!");
    const x = 42;
    if (x > 40) {
        console.log("x is greater than 40");
    }
}

helloWorld();
```

### 言語指定なし

```
これは言語指定なしのコードブロックです。
インデントされたテキストとして表示されます。
複数行にわたるコードもここに書けます。
```

### 長いテキストの折り返しテスト

これは非常に長い段落です。markdown-viewerのテキスト折り返し機能がどのように動作するかをテストするために書いています。このテキストは意図的に長くしていますが、意味のある内容にするのは難しいですね。とにかく、折り返しが正しく動作し、単語の途中で切れないことを確認します。また、ウィンドウサイズが変更されたときにも適切に再レイアウトされることを期待しています。
